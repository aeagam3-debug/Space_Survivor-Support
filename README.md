# Star Survivor — Support

This repository hosts the official support page for **Star Survivor**, a space-themed survivor game for iOS and Android.

## About the Game

Star Survivor is a one-thumb arcade survivor game where you pilot a lone craft against endless waves of alien drones, swarmers, and motherships. Drag anywhere on screen to fly — weapons fire automatically. Collect energy shards to level up, unlock new weapons, and chase a longer run.

**Features:**
- 6 weapons × 5 upgrade tiers each
- 7 passive upgrades for run-to-run specialization
- 11 permanent meta-upgrades unlocked with Stardust
- Boss fights every 90 seconds
- Random horde events that intensify over time
- 6 languages: English, German, Spanish, French, Italian, Turkish
- Customizable music & SFX volume
- Free to play
